# Visual Design

## 文書の位置づけ

この文書には現在の絵作りに関する設計だけを記載する。過去の案や議論の経緯は含めない。

現段階では設計途中であり、実装や確認用プロトタイプは作成しない。

## 1. 世界観と空間

深海をモチーフとする。

マリンスノーを画面内の視覚要素として採用する。具体的な量、動き、表現方法は未決定。

## 2. 動きのモチーフ

- 新体操のリボン演技
- 映画『マトリックス』のセンチネルの付属肢に見られるうねり

## 3. 質感

- メタリックな質感を好む
- チープに見える金属表現は避ける

## 4. 避けたい見え方

- Snakeゲームのような追従運動
- 直線を単純に揺らしただけに見える表現

## 5. 未決事項

金属の種類、色、表面仕上げ、反射の性質、具体的な形、構図、空間、カメラ、ライティングは未決定。
