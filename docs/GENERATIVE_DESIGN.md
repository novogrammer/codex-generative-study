# Generative Design

## 文書の位置づけ

この文書には現在の設計だけを記載する。過去の案や議論の経緯は含めない。

現段階では設計途中であり、実装、擬似コード、確認用プロトタイプは作成しない。

## 1. 中心となる生成原理

過去に採用した次の2D生成方法を、今回は3Dへ適用する方向で検討する。

- 要素をY方向に並べる
- 2Dノイズの1D部分を使い、時間 `t` で標本位置をずらす
- 各要素をX方向へ変位させる
- XをYの関数として一意に求める
- 傾きを `dx/dy` から求める

3Dでの具体的な生成方法はまだ決定していない。

## 2. システムを構成する要素

- チューブ状の表面を構成する断面列
- 断面列を変形させる、より疎な変形フレーム列

断面数と変形フレーム数は分けて扱い、1対1にはしない。具体的な数は未決定。

## 3. 要素同士の関係

根元から親子関係を順番に解く方式では、依存する深さに応じて計算段数が増えるという懸念がある。

変形フレームには親子関係を設けず、それぞれの位置と姿勢を独立に決める。断面は近傍の変形フレームを補間して変形する。

それ以外の接続関係と相互作用は未決定。

## 4. 大域的な振る舞い

3Dへ拡張した結果が、単に直線を揺らしているように見えない方法を探る。

それ以外の大域的な振る舞いは未決定。

## 5. 見た目とモチーフ

現時点で関心が示されている動きは次の通り。

- 新体操のリボン演技
- 映画『マトリックス』のセンチネルの付属肢に見られるうねり

世界観と空間は、深海をモチーフとする。

質感については次の方針とする。

- メタリックな質感を好む
- チープに見える金属表現は避ける

避けたい見え方は次の通り。

- Snakeゲームのような追従運動
- 直線を単純に揺らしただけに見える表現

金属の種類、色、表面仕上げ、反射の性質、具体的な形、構図、空間、カメラ、ライティングは未決定。
